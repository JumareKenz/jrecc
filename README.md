## ğŸš€ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm, yarn, or pnpm package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/JumareKenz/jrecc.git
   cd jrecc
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the website.

### Build for Production

```bash
npm run build
npm start
```

## ğŸ¨ Color Theme

The website uses a carefully curated color palette:

### Primary Colors
- **Impact Green**: `#2E7D32` - Primary brand color for headers, navigation, and CTAs
- **Deep Research Blue**: `#0B3C5D` - Supporting color for text and secondary elements
- **Gold Highlight**: `#F9A825` - Accent color for highlights, stats, and active states

- **Headers/Navbar**: Impact Green

- **Cards**: Soft Grey with white content
